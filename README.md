

# Customer Segmentation Dashboard

## Project Overview
This project analyzes customer data and segments customers based on their demographics and purchasing behavior using Power BI.

## Dataset
- Mall Customers Dataset (CSV)

## Tools Used
- Microsoft Power BI
- CSV Dataset

## Features
- Customer segmentation by gender
- Age group analysis
- Average annual income analysis
- Average spending score analysis
- Interactive slicers for filtering
- KPI cards for customer insights

## Dashboard Visuals
- KPI Cards
- Pie Chart
- Column Chart
- Scatter Chart
- Table
- Slicers

## Key Insights
- Analyzed customer demographics.
- Compared annual income and spending score.
- Visualized customer segments for better business understanding.
- Built an interactive dashboard for data exploration.
